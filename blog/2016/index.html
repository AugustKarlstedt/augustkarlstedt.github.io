Quick setup instructions.

  1. Download the image located here https://www.offensive-security.com/kali-linux-arm-images/

  • Uncategorized
  • Leave a comment

I found out the hard way.

After upgrading my laptop to Windows 10 Version 1607 (Anniversary Update), I was excited to try Bash in Windows. That‚Äôs where my problems started.

After enabling the feature in the ‚ÄúTurn Windows Features On or Off‚Äù dialogue, I rebooted my system and was faced with blue screen stating BitLocker encountered a TPM issue.

Resolution: make sure your BIOS is up-to-date and has TPM enabled, then roll back the latest changes using a recovery drive. Finally, manually enter your BitLocker key to boot into Windows, run tpm.msc, and prepare the TPM.
