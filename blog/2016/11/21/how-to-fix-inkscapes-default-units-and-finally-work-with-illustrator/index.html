    How to Fix Inkscape’s Default Units (and Finally Work With Illustrator)

    • Uncategorized
    • Leave a comment

    For some odd reason, Inkscape defaults to having 90px = 1in (and soon 96px = 1in). Compared to Illustrator, where 72px = 1in, there are some issues converting back and forth between programs.

    There are no configuration settings in Inkscape to change this, but I have found a way to do so anyway.

    Navigate to where you’ve installed Inkscape (for me, this is C:\Program Files (x86)\Inkscape), then open the share folder, then open the ui folder. The folder you should be in is C:\Program Files (x86)\Inkscape\share\ui and you should see a file called units.xml.

    Make a copy of units.xml called units.bak.xml and then open the original units.xml in a text editor.

    Change your file from

    All of your units now will calculate correctly! As a test, create a box with size 72px by 72px. Changing the units afterward, you’ll see that this is equal to 1in and correct for the other values as well.

    Make sure you also set your default bitmap resolution to 72dpi in Preferences (Shift + Ctrl + P) -> Bitmaps

    Another cool feature is that Inkscape supports em units! However, after reviewing the source, the base font size is fixed at 12px. Maybe I can submit a fix for this, hmm.

    Happy Inkscaping!
